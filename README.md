# Enumeration

Пет-проект, созданный **для удовольствия** и в рамках упражнения по изучению React. В основном я делал его **для себя**, но он может пригодиться и другим людям, которые всерьёз увлекаются интеллектуальными викторинами — в первую очередь игрой **«Что? Где? Когда?»** (ЧГК) — и используют тренировки для отработки навыков.

**Демо:** [vladimino.github.io/enumeration](http://vladimino.github.io/enumeration/)

## О проекте

**Enumeration** (перебор) — это тренажёр, в котором вам предлагается список тем или категорий, а задача — вспомнить и назвать как можно больше подходящих элементов.

- За каждый верный ответ начисляются **очки**
- Можно ставить **рекорды** и **переигрывать** сценарии, чтобы улучшить результат
- **Общее количество очков** отображается на странице профиля
- Доступно **распределение очков по категориям**

Сделано с любовью.

## Разработка

```bash
npm install
npm run dev      # локальный сервер: http://localhost:5173/enumeration/
npm test         # тесты (Vitest)
npm run build    # production-сборка в dist/
npm run preview  # предпросмотр production-сборки
```

Стек: React 19, Vite, React Router, Semantic UI (CSS).
